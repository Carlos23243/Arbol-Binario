# Arbol-Binario
# Árbol Binario de Búsqueda (BST) en C#
Esta es una implementación robusta de un Árbol Binario de Búsqueda (Binary Search Tree) desarrollada en C# para consola. El proyecto demuestra el uso de estructuras de datos dinámicas, recursividad y gestión de nodos.
# Características

El programa incluye un menú interactivo que permite realizar las siguientes operaciones:
-Inserción: Agrega nuevos valores manteniendo la propiedad del BST ($izq < raíz < der$).
-Búsqueda: Localiza valores dentro del árbol de manera eficiente.
-Eliminación: Maneja los tres casos de borrado (nodos hoja, con un hijo o con dos hijos).
-Recorridos: Visualización de elementos en Preorden, Inorden y Postorden.
-Estadísticas: Cálculo de la altura del árbol, valor mínimo y valor máximo.
-Limpieza: Opción para vaciar el árbol por completo.
